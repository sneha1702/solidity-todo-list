# solidity-todo-list

Steps to launch the application: 

1. initiate ganache cli, type command 'ganache-cli'

2. login in metamask with the passphrase from the ganache-cli account, and select the active account to the localhost.

3. deploy truffle smart contracts, with command truffle deploy

4. to launch the frontend application, run 'npm run dev'

5. follow through the metamask popups
